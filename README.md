# Registration Form Task – Clean Code

## Task Description
Create a **registration form** that collects user information (e.g., name, email, password) and follows **10 clean code principles**. The form can be a web form, console application, or GUI.

## Requirements
1. Implement a registration form that:
   - Collects at least **name, email, and password**.
   - Validates user input (e.g., proper email format, password rules).
   - Provides meaningful error messages.

2. Follow **10 clean code principles**, such as:
   1. Meaningful and descriptive names.
   2. Small and focused functions.
   3. Avoid duplication (DRY principle).
   4. Clear and concise comments.
   5. Consistent formatting and style.
   6. Handle errors gracefully.
   7. Keep code modular.
   8. Avoid magic numbers or strings.
   9. Use readable logic and control structures.
   10. Organize files and code logically.
